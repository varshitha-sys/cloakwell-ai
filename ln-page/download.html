Download & setup

Clone the repo and run the stack with Docker Compose.

  1. Clone

    git clone https://github.com/varshitha-sys/amd-hackathon-track3.git
cd amd-hackathon-track3

  2. Start services

    docker compose up

  3. Load the browser extension

    Chrome → Extensions → Developer mode → Load unpacked → select the extension/ folder in this repo.
